My time in the NHS

I worked in a trust with some brilliant people, but I was still pretty new to all of this.

One of my biggest frustrations was being asked questions that I just didn’t know the right way to answer… but I was sure someone out there would have thought about this before.

These generally came under a few categories. - What’s going to happen to our demand? - What’s going to happen to our waiting lists? - Are we going to meet target x?

And then there were also lots of better questions I wanted to be able to answer.

  • What can we do to fix this?
  • Who aren’t we serving?

And you can’t look at that list and tell me we don’t all have these same questions.

We have different systems, and different pathways, but I’m far from the first person to be standing up here yelling about having the same problems and repeated solutions.

And it actually makes me a little mad because we shouldn’t, in 2025, be experiencing a postcode lottery of whether our trust has a data science team, and advanced analytics capabilities. I know if I was treated in trust X it’s by a team who are making advanced predictions of bed demand, and

What does the shift need to be?

Shared Licenced Documented Tested Versioned Maintained Collaborative Advertised

And packaged?

Shared = on GitHub Licenced = an appropriate licence so people can reuse it without getting into trouble Documented = clear documentation on how to use it Tested = Actual tests with a framework Versioned = Make actual versions so you know what you’re getting into with updates Maintained = You Collaborative = designed for collaboration, with issues and a code of conduct Advertised = let people know it exists Packaged or parameterisable = installable code OR code that is designed to allow people to input their own data in some way

My final call

We don’t just need to get things into packages.

We need to get them out there into the hands of the people who need them.

There are so many parts to that puzzle slowly going together, but I still don’t feel like I have that ‘magic book’ that I felt must exist when I started my NHS journey.

So I’ve begun to write that.
